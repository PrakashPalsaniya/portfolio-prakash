# Prakash Palsaniya — Portfolio

A personal portfolio inspired by GitHub’s profile experience, built to showcase my work in backend engineering, AI-native applications, and distributed systems.

🔗 **Live site:** [prakash-portfolio.vercel.app](https://prakash-portfolio.vercel.app)

## ✨ Highlights

- GitHub-style UI with clean dark/light theming
- Project-first storytelling with architecture and outcome focus
- Dedicated sections for profile, featured work, skills, activity, and contact
- Fully responsive, fast, and deployed on Vercel

## 🛠 Tech Stack

- **Frontend:** React 18, Vite
- **Styling:** Plain CSS (custom properties + GitHub-inspired design system)
- **Animation:** Framer Motion
- **Deployment:** Vercel

## 📌 Featured Projects

- **DocuMind** — Agentic RAG chat-with-PDF platform (LangGraph + Qdrant)
- **Finmate** — AI expense tracker with assistant-driven insights (MERN + Gemini)
- **Screenly** — Real-time movie ticketing platform (Socket.IO + RabbitMQ)
- **PKLinks** — URL shortener designed as microservices

## 🚀 Getting Started

### Prerequisites

- Node.js (18+ recommended)
- npm

### Installation

```bash
npm install
```

### Run in development

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```

## 📂 Project Structure

```text
src/
  components/   # UI building blocks (header, profile, pinned, activity, etc.)
  data.js       # Profile, projects, skills, and timeline content
  App.jsx       # Main page composition
  index.css     # Global styles and theme variables
```

## 🌐 Deployment

The app is deployed on **Vercel**.  
`vercel.json` configures SPA routing and static asset behavior.
